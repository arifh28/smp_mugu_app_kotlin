Membuat Aplikasi Localhost Dapat Diakses oleh Public dengan Ngrok

Ngrok adalah salah satu tools untuk membuat aplikasi localhost yang biasanya hanya dapat diakses melalui komputer kita, dapat diakses melalui jaringan internet dan diakses oleh publik. Ngrok memiliki banyak fitur dan sangat mudah digunakan.

Anda dapat menggunakan Ngrok untuk keperluan testing dan sharing aplikasi Anda dengan orang lain. Dalam artikel ini, kami akan memberikan tutorial cara menggunakan Ngrok untuk membuat aplikasi localhost dapat diakses oleh publik.

Langkah-langkahnya adalah sebagai berikut:

  1. Download dan install Ngrok di komputer Anda.
  2. Buka terminal atau command prompt, lalu jalankan perintah berikut:

    3. ngrok http 80

  3. Ngrok akan men-generate URL publik yang dapat Anda gunakan untuk mengakses aplikasi localhost Anda.
  4. Salin URL publik tersebut dan bagikan kepada orang lain untuk mengakses aplikasi localhost Anda.

Itulah cara mudah untuk membuat aplikasi localhost dapat diakses oleh publik dengan Ngrok. Semoga bermanfaat!
